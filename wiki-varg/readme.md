# मराठी विकिपीडिया

OSM वरील माहितीची पडताळणी करायला [विकिपीडियाचा](https://mr.wikipedia.org/wiki/%E0%A4%AE%E0%A5%81%E0%A4%96%E0%A4%AA%E0%A5%83%E0%A4%B7%E0%A5%8D%E0%A4%A0) वापर करता येतो.

ह्या फोल्डर मध्ये विकिपीडिया वरून गोळा केलेली काही उपयुक्त माहिती साठवली आहे.

यादी मिळवण्याची पद्धत -

विकिपीडिया वरील कोणत्याही वर्गातील सर्व पृष्ठांची यादी मिळविण्यासाठी -
https://petscan.wmflabs.org/

उदा. - _वर्ग:महाराष्ट्र राज्यातील शहरे व गावे_ ह्या वर्गात २०,००० पेक्षा जास्त पृष्ठ आहेत. त्यांची यादी डाउनलोड करण्यासाठी -

खालील प्रमाणे सेटिंग करा, व "Do it" बटण दाबा. 

```
Categories page :
  Language = mr
  Categories = महाराष्ट्र राज्यातील शहरे व गावे

Output page :
  Format = plain text (किंवा इतर कोणतेही)
  Sort = By title
```

अनेक वर्ग एकत्र शोधण्यासाठी, प्रत्येक ओळीवर एक वर्गाचे नाव लिहा. जसे की,
```
महाराष्ट्रातील_शहरे
महाराष्ट्रातील_तालुके
महाराष्ट्र राज्यातील शहरे व गावे
```

इथे साठवलेले याद्या -

```
mh-shahare-gaave.txt
mh-shahar-taluke-gaave.txt
mh-taluka-v-shahare.txt
```
